# Multi-dimensional Chaos I: Classical and Quantum Mechanics

This page indexes supplementary material to the paper <a href=https://arxiv.org/abs/2510.03007>arXiv:hep-th/2510.03007</a>.

* The [Quantum pinball Julia notebook](../code/Julia/Quantum%20pinball.ipynb) contains the code to compute the scattering amplitude for the two- and three-disk quantum pinball analyzed in the paper. It is written to be generalizable to the n-disk system as well.
