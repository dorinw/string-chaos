# Multi-dimensional Chaos II: String scattering amplitudes, curve repulsion, and RMT

This page indexes the supplementary material associated with the paper <a href=https://arxiv.org/abs/2606.24490>arXiv:hep-th/2606.24490</a>.

We uploaded the following code:

* [HTTT curves analysis.nb](../code/Mathematica/HTTT%20curves%20analysis.nb) is a Mathematica notebook used for all the main computations in the paper. It is fully self-contained.

* [HTTT python.ipynb](../code/Python/HTTT%20python.ipynb) is a notebook containing an implementation of the same formulas of the HT->TT amplitude in Python. We have used it mainly for plotting, but it can be expanded to carry out a full numerical analysis.

The data, results of the computations carried out by the Mathematica notebook, will be uploaded separately on Zenodo.
