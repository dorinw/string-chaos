# Multi-dimensional Chaos II: String scattering amplitudes, curve repulsion, and RMT

This page indexes the supplementary material associated with the paper <a href=https://arxiv.org/abs/2606.xxxxx>arXiv:hep-th/2606.xxxxx</a>.

We uploaded the following code:

* [HTTT curves analysis.nb](code/Mathematica/HTTT%20curves%20analysis.nb) is a Mathematica notebook used for all the main computations in the paper.
* [HTTT amplitude.ipynb](code/Python/HTTT%20amplitude.ipynb) is a notebook containing an implementation of the same formulas of the HT->TT amplitude in Python. We have used it mainly for plotting, but it can be expanded to carry out a full numerical analysis.
